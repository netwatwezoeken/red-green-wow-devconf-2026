# FizzBuzz Code Kata

This is a FizzBuzz code kata starter project. Read about the kata here: https://kata-log.rocks/fizz-buzz-kata

## Project Structure

- **FizzBuzz**: `FizzBuzz/FizzBuzz.cs` is the class to further extend.
- **FizzBuzzTests**: `FizzBuzzTests/FizzBuzzTests.cs` contains one test to get started.
